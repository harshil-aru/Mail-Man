# Mail-Man
A real time mail client which can read your mails and can manage them by automatically unsubscribing to the mails which are of no use to use!!!1
this is a real time gmail web client which can fetch your mails and subscribe and unsubscribe a real time 
it basically uses oAuth 2.0 to get web token from gmail and helps you easily communicate with gmail API
libraries used- 
- Express
- Passport
- Gmail Api
- Google Api
- HTTP server
- node-google-api
- bodyparser
- session
- MongoDb
